# README

Contact is a Contact management app for university Library. It is build using Ruby on rails and SQLite.

## Prerequisite
 * Ruby 2.5+
 * Rails 5+

## Installation
```
$ git clone https://github.com/AkhilGKrishnan/ContactManagement.git
$cd ContactManagement
$ bundle install
$ rake db:create
$ rake db:migrate
$ rails s

```
